Add `sectionGroups` prop to `admin-settings-page` — render settings sections under categorical eyebrow labels (e.g., Account, Workspace, Venue) instead of a flat list. Each group supports a meta badge and a muted flag for de-emphasized sections. Existing `sections` prop usage is unchanged. Install via `npx visor add admin-settings-page`.
