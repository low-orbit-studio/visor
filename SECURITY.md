# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.x     | Yes       |

## Reporting a Vulnerability

**Please do not report security vulnerabilities via public GitHub issues.**

To report a vulnerability privately, email **justin@loworbit.studio** with the subject line `[SECURITY] Visor — <brief description>`.

### What to include

- Affected version(s)
- Step-by-step reproduction instructions
- Impact assessment (what an attacker could do)
- Any suggested mitigations (optional)

### Response timeline

This project is maintained by a single person. Best effort applies — we aim to acknowledge reports within **48 hours** and will keep you informed as the issue is investigated and resolved.

Thank you for helping keep Visor and its users safe.
