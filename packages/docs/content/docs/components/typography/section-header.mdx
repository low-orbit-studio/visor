## Overview

`SectionHeader` is a simple heading row for sectioning content within a page.
It renders a title, an optional subtitle, and an optional trailing slot for
actions, counts, or links.

## Installation

```bash
npx visor add section-header
```

## Usage

```tsx
import { SectionHeader } from '@/components/ui/section-header/section-header';

<SectionHeader title="Recent activity" />

<SectionHeader
  title="Recent activity"
  subtitle="Last 30 days"
/>

<SectionHeader
  title="Recent activity"
  subtitle="Last 30 days"
  trailing={<a href="/activity">View all</a>}
/>
```

## Installation

```bash
npx visor add section-header --target flutter
```

## Usage

```dart
import 'package:ui/ui.dart';

VisorSectionHeader(
  title: 'Recent activity',
)
```

## With Subtitle

```dart
VisorSectionHeader(
  title: 'Recent activity',
  subtitle: 'Last 30 days',
)
```

## With Trailing Action

```dart
VisorSectionHeader(
  title: 'Recent activity',
  subtitle: 'Last 30 days',
  trailing: TextButton(
    onPressed: _viewAll,
    child: const Text('View all'),
  ),
)
```

See the [Flutter Section Header](/docs/flutter/widgets/section-header) page for the full API reference.
