## Installation

```bash
npx visor add phone-input
```

This copies two files into your project:

- `components/ui/phone-input/phone-input.tsx` — the component
- `components/ui/phone-input/phone-input.module.css` — the styles

This also installs the `input` component as a dependency.

## Usage

```tsx
import { PhoneInput } from '@/components/ui/phone-input/phone-input';

<PhoneInput placeholder="(555) 123-4567" />
```

## Accessibility

- Renders a native `<input>` with `type="tel"` and `inputmode="tel"` for appropriate mobile keyboard
- Always pair with a `<Label>` using `htmlFor`/`id`, or provide `aria-label` directly
- The `onChange` callback returns raw digits only — display formatting is applied visually but not stored in the value
- For international numbers, consider pairing with a country code selector
