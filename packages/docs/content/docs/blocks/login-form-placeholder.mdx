## Installation

```bash
npx visor add --block login-form-placeholder
```

This copies files into your project:

- `blocks/login-form-placeholder/login-form-placeholder.tsx` — the block component
- `blocks/login-form-placeholder/login-form-placeholder.module.css` — the styles

## Usage

```tsx
import { LoginFormPlaceholder } from '@/blocks/login-form-placeholder/login-form-placeholder';

export default function LoginPage() {
  return <LoginFormPlaceholder />;
}
```

## About Blocks

Blocks are complete UI patterns made up of multiple Visor components. Unlike individual components, blocks represent larger, composed sections of UI — such as login forms, settings pages, or dashboard panels.

Blocks are copy-and-own, just like components. Install them into your project and customize freely.
